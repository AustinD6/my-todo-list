这是一个基于 **React + TypeScript** 构建的高颜值、功能完备的待办事项管理应用。采用了现代化的 **Glassmorphism（玻璃拟物化）** 设计风格，并支持作为 **PWA** 或 **原生 App** 安装使用。

🌐 **[立即在线预览](https://austind6.github.io/my-todo-list/)**

---

## ✨ 功能亮点

- 🎨 **视觉艺术**：精美的玻璃磨砂质感界面，适配各种背景。
- 📝 **任务管理**：
  - **基础操作**：添加、快速切换状态、一键删除。
  - **实时编辑**：点击铅笔图标即可直接修改任务内容。
  - **搜索功能**：强大的实时关键词搜索，任务再多也能秒找。
- 🛠️ **高级属性**：
  - **分类系统**、**优先级区分**、**截止日期提醒**。
- 💾 **本地存储**：数据自动保存至浏览器 `LocalStorage`，支持离线查看。
- 📱 **多端支持**：
  - **PWA**：支持“添加到主屏幕”，像原生 App 一样全屏运行。
  - **App 准备**：已集成 Capacitor 架构，支持导出为 Android/iOS 原生包。

---

## 📲 手机安装教程 (PWA)

1. 在手机浏览器打开 [预览链接](https://austind6.github.io/my-todo-list/)。
2. **iOS (Safari)**: 点击“分享”按钮 -> 选择 **“添加到主屏幕”**。
3. **Android (Chrome)**: 点击右上角菜单 -> 选择 **“安装应用”** 或 **“添加到主屏幕”**。
4. 现在你可以从桌面直接启动，享受无边框的全屏体验！

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite + Vite PWA Plugin
- **跨平台**: Capacitor (Android Ready)
- **图标**: Lucide React
- **样式**: Vanilla CSS (Custom Glassmorphism)
