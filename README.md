# 🚀 现代感备忘录应用 (Todo List App)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)
![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=for-the-badge&logo=typescript&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white)

这是一个基于 **React + TypeScript** 构建的高颜值、功能完备的待办事项管理应用。采用了现代化的 **Glassmorphism（玻璃拟物化）** 设计风格，旨在提供极致的交互体验。

🌐 **[立即在线预览](https://austind6.github.io/my-todo-list/)**

---

## ✨ 功能亮点

- 🎨 **视觉艺术**：精美的玻璃磨砂质感界面，适配各种背景。
- 📝 **任务管理**：
  - **基础操作**：添加、快速切换状态、一键删除。
  - **实时编辑**：点击铅笔图标即可直接修改任务内容。
  - **搜索功能**：强大的实时关键词搜索，任务再多也能秒找。
- 🛠️ **高级属性**：
  - **分类系统**：支持工作、个人、购物、健康等多种维度归类。
  - **优先级**：高、中、低三种优先级视觉区分。
  - **截止日期**：为每个任务设置目标完成时间。
- 📊 **智能筛选**：支持按状态（全部/待办/已完成）和分类进行过滤。
- 💾 **本地存储**：数据自动保存至浏览器 `LocalStorage`，刷新页面不丢失。

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/AustinD6/my-todo-list.git
cd my-todo-list
```

### 2. 安装依赖
```bash
npm install
```

### 3. 本地运行
```bash
npm run dev
```

### 4. 构建发布
```bash
npm run deploy
```
